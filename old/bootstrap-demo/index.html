BOOTSTRAP - For me Bootstrap, is my saving grace, Seeing this in action gave me the ability to understand how the grid system functions and how flexible its components truly are, Its amazing how easy and quick you can accomplish a responsive, mobile-first website. By design BOOTSTRAP offers consistent design by using re-usable components and give web designers a solid foundation for creating interesting Bootstrap themes.
Open Source

Bootstrap is an open source denoting software for web developers, the code is made freely available and can be copied and modified. Bootstrap comes with many quick enhancements such as :responsive grid system, prebuilt components, and plugins

Mobile First

Equipped with a powerful flex box grid , Bootstrap allows you to set up a layout quickly with the 12 column system and 5 responsive tiers.

Alerts

Alerts provide feedback messages with 8 contexual classes such as:

  1. primary
  2. secondary
  3. success
  4. danger
  5. warning
  6. info
  7. light
  8. dark
Components Library

No JavaScript!! Components library includes buttons and lists to corousels and modals, they are easy to intergrate and add quickly to any site

List Groups

What a great way to display a series of content! List groups are flexible and powerful and can be modified to support any content.
