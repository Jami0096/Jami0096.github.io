BOOTSTRAP - For me Bootstrap, is my saving grace, Seeing this in action gave me the ability to understand how the grid system functions and how flexible its components truly are, Its amazing how easy and quick you can accomplish a simple website within an hour or 2.
Open Source

Bootstrap is an open source denoting software for web developers, the code is made freely available and can be copied and modified. Bootstrap comes with many quick enhancements such as :responsive grid system, prebuilt components, and plugins

Mobile First

Equipted with a powerful flex box grid , Bootstrap allows you to set up a layout quickly with the 12 column system and 5 responsive tiers.
